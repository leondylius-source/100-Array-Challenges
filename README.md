# 📊 100 Array Questions – Mastering Arrays in DSA

Welcome to **100 Array Questions**, a curated collection of hand-picked problems designed to help you master array-based data structures and algorithms. Whether you're preparing for coding interviews or just strengthening your problem-solving skills, this repository has something for everyone.

---

## 🚀 What You'll Find

- ✅ **100 array problems** from basic to advanced  
- 🧠 **Detailed solutions** in Java (or your language of choice)  
- 💡 Concepts such as:  
  - Traversals and searching  
  - Sorting and two-pointer techniques  
  - Sliding window & prefix sums  
  - Hashing, stacks, and more  
- 📈 Difficulty levels: Easy → Medium → Hard

1. Fork/Clone this repo:
